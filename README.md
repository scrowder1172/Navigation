# Project 09: Navigation

## Hacking With Swift project 09

This is Project 09 in Paul Hudson's 100 Days of SwiftUI course. The project demonstrates various ways to implement navigation within iOS apps.

## Course Link

Check out Paul's course for more on SwiftUI: https://www.hackingwithswift.com/100/swiftui

## How to install

1. fork the project or download the .zip
2. open project in XCode v15 or later (earlier versions may or may not compile correctly)
3. run in simulator
